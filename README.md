# EffectsGUI

Plugin para Minecraft que adiciona um menu gráfico para ativar e desativar efeitos de poção.

## Compatibilidade

- Servidor: Paper (recomendado) ou Bukkit/Spigot
- Versão do Minecraft: 1.21 ou superior
- Versão do plugin: 1.1

## Como usar

Execute o comando `/efeitos` para abrir o menu de efeitos. Clique em um efeito para ativá-lo ou desativá-lo.

## Efeitos disponíveis

- Salto
- Queda Lenta
- Velocidade
- Mineração Rápida
- Respiração Aquática
- Visão Noturna
- Resistência ao Fogo

## Permissões

- `effectsgui.effect.*` - Acesso a todos os efeitos
- `effectsgui.effect.jump` - Acesso ao efeito de Salto
- `effectsgui.effect.slowfall` - Acesso ao efeito de Queda Lenta
- `effectsgui.effect.speed` - Acesso ao efeito de Velocidade
- `effectsgui.effect.haste` - Acesso ao efeito de Mineração Rápida
- `effectsgui.effect.waterbreathing` - Acesso ao efeito de Respiração Aquática
- `effectsgui.effect.nightvision` - Acesso ao efeito de Visão Noturna
- `effectsgui.effect.fireresp` - Acesso ao efeito de Resistência ao Fogo

Por padrão, apenas operadores têm acesso aos efeitos.
